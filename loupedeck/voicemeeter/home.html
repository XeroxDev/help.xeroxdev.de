What is this Plugin?

This Loupedeck Plugin allows you to control the VoiceMeeter App

How to use it?

  1. Install the VoiceMeeter App
  2. Install the Plugin
  3. Add Actions to Loupedeck
  4. Have fun controlling your voicemeeter
